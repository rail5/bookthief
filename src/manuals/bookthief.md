# NAME

bookthief - Convert PDFs to "pamphlet" style for home printing

# SYNOPSIS

```bash
bookthief input.pdf
```

# DESCRIPTION

BookThief is a graphical front-end for Liesel, a tool that prepares PDFs to be DIY printed at home in "pamphlet" style.

BookThief provides an easy-to-use interface for selecting input files and configuring options without needing to use the command line.
